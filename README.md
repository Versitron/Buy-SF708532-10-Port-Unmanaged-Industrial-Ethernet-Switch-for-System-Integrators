# Versitron SF708532 | 10-Port Unmanaged Industrial Ethernet Switch for System Integrators  

The Versitron SF708532 is a rugged 10-port unmanaged industrial Ethernet switch designed for system integrators requiring reliable, high-performance connectivity in automation, surveillance, and control networks. Featuring 8 Fast Ethernet RJ45 ports and 2 multimode ST fiber ports, it enables seamless copper-to-fiber integration for long-distance and interference-free communication.
